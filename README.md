## Installation

Este proyecto requiere [Node.js](https://nodejs.org/) v10 para funcionar.
```sh}
Clonar este repositorio 
cd entregas
npm start
```

Son necesarias las siguientes dependencias:
React-Router-Dom, ChakraUI
